## 心得

這次的作業讓我了解到在寫出一個好看好用的網頁，除了需要考慮到排版以外，還需要考慮網頁的效能。
網頁效能如果不好，將會導致使用者在使用上會有不好的體驗，而且也有可能會因為裝置較差或者網路不好導致網頁跑不出來或是跑不動。
希望以後在寫網頁是能夠在第一時間就去考慮到這些東西，如此就不用花太多時間去做效能檢測。
